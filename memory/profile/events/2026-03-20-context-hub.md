# Context Hub 技能安装

## 来源
https://github.com/andrewyng/context-hub（吴恩达团队）

## 安装
```bash
npm install -g @aisuite/chub
```

## 技能位置
`skills/context-hub-1.0.0/SKILL.md`

## 功能
通过 `chub` CLI 获取第三方库/SDK/API 的最新文档，避免 AI 编程代理的 API 幻觉和过时信息

## 验证
成功搜索 `openai/chat` 并获取 Python 文档
