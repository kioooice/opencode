### OpenClaw Cron

- Gateway 内置调度器，支持 `openclaw cron add` 创建任务
- 支持 one-shot (`--at`) 和 recurring (`--cron`) 调度
- 支持 main session (system event) 和 isolated session (agent turn)
- 配置存储：`~/.openclaw/cron/jobs.json`

## 反思候选

<!-- 心跳时可能提取的洞察候选 -->

1. **记忆系统的价值在于"提炼"而非"存储"**——原始日志再详细，不提炼就无法检索
2. **标签系统要尽早启用**，否则后期补标签成本很高
3. **memory_search 需要 OpenAI API Key**——这是启用语义搜索的前置条件，记录在 insights 中避免重复调查
4. **Windows 安装 bash 脚本**：下载 tar.gz → tar 解压 → 手动执行等效 PowerShell 操作
5. **OpenClaw cron 配置**：通过 `~/.openclaw/cron/jobs.json` 配置，需要 Gateway 运行才能生效
