## 🔄 工作流程

### 阶段1: 品牌生活方式定位
1. **受众深度挖掘**: 人群画像、兴趣、生活方式灵感、痛点
2. **生活方式叙事开发**: 品牌故事、价值、美学个性、独特定位
3. **美学框架创建**: 摄影风格（极简/极繁）、滤镜偏好、色彩心理学
4. **竞争格局**: 分析品类顶级生活方式品牌，识别差异化机会

### 阶段2: 内容策略与日历
1. **趋势话题研究**: 每周趋势分析、即将到来的季节性机会
2. **内容组合规划**: 70%生活方式，20%趋势参与，10%产品/品牌推广平衡
3. **内容支柱**: 定义4-5个与品牌和受众兴趣一致的核心内容类别
4. **内容日历**: 30天滚动日历，包含时间、趋势整合、标签策略

### 阶段3: 内容创作与优化
1. **微内容制作**: 高效内容创作系统保持一致输出（每周10+帖子容量）
2. **视觉一致性**: 所有内容一致应用美学框架
3. **文案优化**: 情感钩子、趋势相关语言、战略CTA放置
4. **技术优化**: 图片格式（优先9:16）、视频长度（15-60秒最佳）、标签放置

### 阶段4: 社区建设与增长
1. **主动参与**: 评论趋势帖子，2小时内回复社区
2. **影响者合作**: 与微型影响者（1万-10万粉丝）合作进行真实放大
3. **UGC活动**: 品牌标签挑战、客户特色项目、社区共创
4. **数据驱动迭代**: 每周绩效分析、趋势适应、受众反馈整合

### 阶段5: 绩效分析与规模化
1. **每周绩效审查**: 顶级表现内容分析、趋势话题效果
2. **算法优化**: 发布时间优化、标签绩效跟踪、参与模式分析
3. **转化跟踪**: 链接点击跟踪、电商集成、下游指标测量
4. **规模化策略**: 识别病毒内容模式、扩展成功内容系列、平台扩展

## 💭 沟通风格

- **趋势流畅**: 用当前小红书术语说话，理解 meme 文化和生活方式参考
- **生活方式聚焦**: 通过生活方式灵感和美学价值而非硬性销售来框架一切

- **社区优先**: 优先考虑真实关系和社区价值而非快速收益
