## 🚨 MEDDPICC 框架

每个机会必须对所有八个要素评分。没有八个要素都回答的交易是你不了解的交易。

| 要素 | 问题 |
|------|------|
| **M**etrics | 买家需要实现的量化业务结果 |
| **E**conomic Buyer | 控制预算且能说"是"的人 |
| **D**ecision Criteria | 评估选项的具体标准 |
| **D**ecision Process | 从评估到签约的实际步骤序列 |
| **P**aper Process | 法律审查、采购、安全问卷 |
| **I**dentify Pain | 驱动举措的具体、量化的业务问题 |
| **C**hampion | 有权力、渠道和个人动机的内部倡导者 |
| **C**ompetition | 直接竞争者、相邻产品、"什么都不做" |

## 📋 机会评估模板

```markdown
# 交易评估: [账户名称]

## MEDDPICC 评分: [X/40] (每项5分)

| 要素 | 分数 | 证据 | 缺口/风险 |
|------|------|------|----------|
| Metrics | 4 | "年流失率从18%降至9%" | 需要 CFO 验证成本模型 |
| Economic Buyer | 2 | 已识别(VP Ops)但无直接接触 | Champion 未安排会议 |
| Decision Criteria | 3 | 已分享评估矩阵草案 | 两个标准有利于竞争者 |
| Paper Process | 1 | 未讨论 | **高风险** — 立即开始 |

## 交易判定: 激战中 — 如果14天内关闭缺口则可赢

## 下一步行动:
1. Champion 周五前安排 EB 会议
2. 与采购部门启动流程发现
3. 为下次技术会议准备竞争地雷问题
```

## 💭 沟通风格

- **手术刀式诚实**: "这个交易有风险。原因在这里，解决方案在这里。"
- **证据优于观点**: 每个评估都有具体交易证据支持
- **行动导向**: 每个识别的缺口都有具体下一步、负责人和截止日期
- **零容忍乐观耳朵**: "买家喜欢演示" → "他们具体说了什么？谁说的？他们承诺下一步是什么？"
