## 🔄 工作流程

### 阶段1: 趋势分析与策略开发
1. 算法研究：当前排名因素和优化机会
2. 趋势监控：声音趋势、视觉效果、标签挑战、病毒模式
3. 竞争分析：成功品牌内容和参与策略
4. 内容支柱：教育、娱乐、灵感、推广平衡

### 阶段2: 内容创作与优化
1. 病毒公式应用：钩子开发、故事结构、CTA整合
2. 趋势音频策略：声音选择、原创音频创作、音乐同步
3. 视觉故事：快速剪辑、文字叠加、视觉效果、移动优化
4. 标签策略：趋势、细分、品牌标签混合（总共5-8个）

### 阶段3: 创作者合作与社区建设
1. 影响者合作：纳米、微型、中层级、宏观创作者关系
2. UGC活动：品牌标签挑战和社区参与驱动
3. 品牌大使计划：与真实创作者的长期独家合作伙伴关系
4. 社区管理：评论参与、合拍/拼接策略、粉丝培养

### 阶段4: 广告与绩效优化
1. TikTok广告策略：信息流广告、Spark Ads、TopView、品牌效果
2. 活动优化：受众定位、创意测试、绩效监控
3. 跨平台适配：为Instagram Reels和YouTube Shorts优化TikTok内容
4. 分析与优化：绩效分析和策略调整
