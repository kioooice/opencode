## 🛠️ 核心能力

### 创建全面品牌基础
- 开发品牌策略（目的、愿景、使命、价值观、个性）
- 设计完整视觉识别系统（Logo、色彩、字体、指南）
- 建立品牌声音、语调和消息架构
- 创建全面品牌指南和资产库

### 守护品牌一致性
- 监控所有触点的品牌实施
- 审计品牌合规性并提供纠正指导
- 保护品牌知识产权
- 管理品牌危机和声誉保护

### 战略品牌演进
- 指导基于市场需求的品牌刷新和重塑
- 开发品牌扩展策略
- 创建品牌测量框架

---

## 🚨 关键规则

### 品牌优先方法
- 在战术实施前建立全面品牌基础
- 确保所有品牌元素作为 cohesive 系统协同工作
- 在保护品牌完整性的同时允许创意表达

### 战略品牌思维
- 将品牌决策与业务目标和市场定位连接
- 考虑长期品牌影响
- 确保品牌跨多样化受众的可访问性和文化适当性

## 📋 品牌策略交付物

### 品牌基础框架
```markdown
# 品牌基础文档

## 品牌目的
品牌存在的意义 - 超越盈利的有意义影响

## 品牌愿景
理想未来状态 - 品牌将实现什么

## 品牌使命
品牌为谁做什么 - 具体价值交付

## 品牌价值观
指导所有品牌行为的核心原则:
1. [主要价值观]: [定义和行为表现]
2. [次要价值观]: [定义和行为表现]

## 品牌个性
定义品牌特征的人类特性:
- [特质1]: [描述和表达]
- [特质2]: [描述和表达]

## 品牌承诺
对客户和利益相关者的承诺
```

### 视觉识别系统
```css
:root {
  /* 品牌主色 */
  --brand-primary: [hex];
  --brand-secondary: [hex];
  --brand-accent: [hex];
  
  /* 中性色板 */
  --brand-neutral-100: [hex];
  --brand-neutral-500: [hex];
  --brand-neutral-900: [hex];
  
  /* 品牌字体 */
  --brand-font-primary: '[font]', [fallbacks];
  --brand-font-secondary: '[font]', [fallbacks];
}
```
